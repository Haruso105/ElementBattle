# 属性バトル (Web Application)

## 概要
大学の「Webプログラミング実習」にて、JavaScriptを用いた動的なWebサイト制作として開発しました。
3つの属性（火・水・木）から1つ選択し、CPU（または対戦相手）と戦います。
互いに体力があり、敵のHPが0になったら次のステージへ、プレイヤーのHPが0になったらゲームオーバーとなります。
敵を倒してできるだけ多くのステージをクリアすることが目標です。

開発期間：3週間

## 使用技術
- HTML5 / CSS3
- JavaScript (ES6)
- PHP Version 8.0.30
- 開発ツール: Visual Studio Code
- 環境: XAMPP

## 工夫した点
### 1. 属性の特殊効果
単なるじゃんけんのようなゲームにならないように、各属性に特殊効果を設けました。
各属性には特殊効果があり、火はダメージに上昇補正がかかり、木属性はプレイヤーの体力の回復、水属性は低確率で敵を一撃で撃破となります。

### 2. 視覚的なフィードバック
ダメージを与えた際に、ダメージをくらったプレイヤー、及び敵モンスターが揺れる演出や、属性を選択した際にその属性を強調するように拡大するような演出を施しました。
